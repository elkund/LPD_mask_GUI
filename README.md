# LPD_mask_GUI
Jupyter-based Interactive masking utility for LPD1M detector. See screenshot.
![Screenshot](/screenshot.png?raw=true "Screenshot")

Red pixels are masked out in the pipeline, magenta pixels are . Left click to mask, right click to unmask. Select masking unit (chip, tile, module) from drop down. You can create a mask per train/pulse or a static mask, identical for every train/pulse.

Written by Tobias Eklund and Yizhi Lui (2025) for FXE beamtimes 3273, 7944.
